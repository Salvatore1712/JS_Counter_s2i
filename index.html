COME FUNZIONA

addEventListener è un metodo di JavaScript usato per associare eventi agli elementi del DOM, come click, input o pressione di tasti. Permette di eseguire una funzione (callback) quando l’evento si verifica, senza sovrascrivere eventuali altri gestori. A differenza di onclick, consente di aggiungere più listener allo stesso elemento e offre maggiore controllo tramite opzioni avanzate.
